# Design System - Cowly

## Brand Direction

Cowly must feel modern, premium, bold, and high-conversion.

The interface should create a strong first impression and feel visually impressive without becoming noisy or hard to use.

The experience should feel polished, fast, and memorable.

## Visual Style

* Modern startup aesthetic
* Premium dark mode support
* Strong contrast
* Spacious layouts
* Clean typography
* Bold visual hierarchy

## Color Palette

Primary:

* Orange: #F97316
* Orange Hover: #EA580C

Neutrals:

* Black: #0B0B0B
* White: #FFFFFF

Support:

* Light background: #F9FAFB
* Border: #E5E7EB
* Dark card: #111111
* Dark elevated card: #1A1A1A

## Dark Mode

Dark mode is not optional. It must be supported from the beginning.

Dark mode should feel premium, not just inverted.
Use deep dark backgrounds, subtle contrast, and preserve orange as the action color.

## Typography

* Use Inter or Geist
* Strong headlines
* Clean body text
* Clear hierarchy
* High readability

## Motion Principles

Animations must feel premium and intentional.

Use:

* smooth entrance animations
* fade + slide combinations
* subtle scale effects
* hover transitions
* soft parallax when appropriate
* staggered reveal for lists/cards
* smooth page transitions if possible

Avoid:

* random animations
* excessive bouncing
* distracting motion
* animations that slow down usability

## Desired Experience

The UI should feel “jaw-dropping” on first impression while still being usable and conversion-focused.

It should feel closer to a premium SaaS landing page than a generic corporate website.

## Components

### Buttons

* Orange primary button
* Strong hover state
* Slight scale or lift on hover
* Rounded corners
* Smooth transitions

### Cards

* Soft shadow
* Rounded corners
* Clean spacing
* Subtle hover motion
* Elevated feel in dark mode

### Sections

* Large spacing
* Strong headings
* Clear visual separation
* Optional gradient or glow accents used sparingly

## Hero Section

The hero must be visually strong.

It should include:

* bold headline
* strong CTA
* animated visual area
* premium spacing
* motion that makes the first screen feel alive

## UX Rules

* Keep flows simple
* Always prioritize clarity
* Animation should enhance, not confuse
* Design must remain responsive and mobile-friendly

## Do Not

* Do not make the interface look generic
* Do not use too many colors
* Do not overuse glassmorphism
* Do not create cluttered layouts
* Do not sacrifice usability for visual effects
