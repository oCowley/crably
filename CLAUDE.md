# Crably

## Overview

Crably is a platform for selling ready-made website products with a fast ecommerce flow and internal project management.

The system has three main layers:

1. Public website
2. Ecommerce/product system
3. Internal admin/project management

## Core Idea

Users can browse website templates, purchase them online, and track project progress.

The internal team manages projects and assigns work to developers.

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* Firebase Auth
* Firestore
* Stripe
* Vercel

## MVP Features

* Landing page
* Product catalog
* Product detail page
* Authentication (Firebase)
* Stripe checkout
* Orders system
* Customer dashboard
* Admin dashboard

## Product Rules

* Products have fixed scope
* No custom development in MVP
* Each order is tied to a template
* Limited customization
* No unlimited revisions

## Development Guidelines

* Keep code simple and modular
* Avoid over-engineering
* Use TypeScript strictly
* Prefer server-side logic for payments
* Use Firebase best practices

## UI Guidelines

* Clean and modern
* Use orange as primary action color
* Support dark mode
* Focus on conversion

## Roles

* customer
* admin
* developer
